
### Merge Sort(분할 정복 알고리즘)

하나의 리스트를 두 개의 균등한 크기의 리스트로 분할을 한 후에 부분 리스트를 합치면서 정렬을 합니다.

그래서, 전체가 정렬이 되게 하는 방식입니다.

주로 재귀함수에서 쓰입니다.

시간복잡도는  O(NlogN)
