### Binary Search

오름차순으로 정렬되어있는 리스트 내에서 특정한 값의 인덱스를 찾는 알고리즘입니다.

정렬된 리스트에서만 사용되기에 빠른 속도이고 시간복잡도는 O(logN)입니다.

n/2 혹은 (n+1)/2 번째를 기준으로 좌우를 비교하면서 원하는 값을 찾는 방식입니다.
