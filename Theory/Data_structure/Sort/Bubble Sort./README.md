### Bubble Sort

인접한 두 element의 값을 비교하는 것으로 두 값이 정렬되어있지 않다면 위치를 교환

정렬이 완료된 elements를 제외하고 아래 과정의 반복합니다.

항상 정렬이 끝난 다음엔 가장 최고값이 맨 끝인 오른쪽으로 이동이 된 상태입니다. 

이러한 이유로 시간 복잡도는 O(N^2)입니다.
